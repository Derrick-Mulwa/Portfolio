Automated website testing using Selenium, Pytest and Page-Object-Model

This is a comprehensive end-to-end automated website testing solution built with Python, leveraging Selenium WebDriver for browser automation and pytest as the testing framework. Structured using the Page Object Model (POM), it ensures scalability, reusability, and ease of maintenance. The framework covers critical user workflows such as registration, login, product browsing, cart operations, and checkout. With centralized configuration via INI files, modular page classes, and detailed HTML reporting, it streamlines regression testing, enhances test accuracy, and significantly reduces manual testing efforts—making it a reliable tool for ensuring web application quality and consistency.
